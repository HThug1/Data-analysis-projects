# Data Analysis Projects

This repository contains three projects focused on real-world business scenarios across operations, risk, and IT support.

---

## 1. Operations Performance Dashboard
Analysed SLA performance, escalation trends, and resolution time across daily operations.

**Key Insight:**  
Workload spikes (>60 cases) increase escalation rates and reduce SLA performance, while lower volumes improve efficiency.

---

## 2. Fraud Detection & Transaction Analysis
Built a rule-based model to identify high-risk transactions based on amount, geography, timing, and behaviour patterns.

**Key Insight:**  
High-value transactions and activity from high-risk regions drive a significant portion of risk exposure.

---

## 3. IT Support Ticket Analysis
Analysed ticket data to evaluate resolution rates, escalation patterns, and issue types.

**Key Insight:**  
High-priority hardware and network issues drive most escalations and longer resolution times, while login issues are resolved quickly.
